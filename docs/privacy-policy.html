🔒 Политика конфиденциальности

WoxAI — AI Voice Recorder & Meeting Assistant

Общие сведения

Настоящая Политика конфиденциальности описывает, как приложение WoxAI (разработчик: AI Studio) собирает, использует и защищает ваши данные при использовании нашего приложения для Android.

Используя приложение, вы соглашаетесь с условиями данной политики.

Какие данные мы собираем

  • Аудиозаписи: голосовые записи, которые вы создаёте в приложении, хранятся локально на вашем устройстве.
  • Транскрипции: текстовые расшифровки ваших записей, создаваемые с помощью AI.
  • Данные аккаунта: email и имя пользователя при авторизации через Google (Firebase Authentication).
  • Профили голоса: пользовательские метки для идентификации участников встреч.
  • Данные об использовании: анонимная статистика для улучшения качества сервиса.

Как мы используем данные

  • Для предоставления основных функций приложения: записи, транскрипции и анализа встреч.
  • Для синхронизации данных между устройствами (при наличии аккаунта).
  • Для улучшения точности AI-расшифровки с использованием Google Gemini API.
  • Для отправки push-уведомлений (Firebase Cloud Messaging).
Мы не продаём и не передаём ваши личные данные третьим лицам в коммерческих целях.

Сторонние сервисы

Приложение использует следующие сторонние сервисы со своими политиками конфиденциальности:

  • Google Firebase — аутентификация, база данных, уведомления.
  • Google Gemini API — AI-транскрипция и анализ текста.
  • Google Play Billing — обработка платежей и подписок.

Разрешения приложения

  • RECORD_AUDIO — для записи голоса и проведения встреч.
  • INTERNET — для синхронизации данных и работы AI-функций.
  • USE_BIOMETRIC / USE_FINGERPRINT — для защиты приложения биометрией (опционально).
  • FOREGROUND_SERVICE — для фоновой записи во время встреч.
  • POST_NOTIFICATIONS — для уведомлений о статусе записи.

Хранение и безопасность данных

Аудиофайлы хранятся на вашем устройстве и, при необходимости, в защищённом облачном хранилище Firebase Storage. Передача данных осуществляется по зашифрованному протоколу HTTPS.

Мы принимаем разумные меры для защиты ваших данных, однако ни один метод передачи данных через интернет не является абсолютно безопасным.

Ваши права

  • Удалить свой аккаунт и все связанные данные в любое время через настройки приложения.
  • Отозвать разрешения приложения через настройки Android.
  • Запросить копию или удаление ваших данных, написав нам на email.

Дети

Наше приложение не предназначено для детей до 13 лет. Мы намеренно не собираем личные данные лиц младше 13 лет.
